# Tomato Paste Craft

A simple Project Zomboid Build 42 mod that adds tomato paste crafting recipes.

## Features

- Craft tomato paste from tomatoes.
- Basic recipe available to all players.
- More efficient recipe unlocked at Cooking level 6.
- Built and tested for Project Zomboid 42.19.0.

## Installation

### Steam Workshop
Subscribe on Steam Workshop and enable the mod in-game.

### Manual
Place the mod folder in your Project Zomboid `mods` directory and enable it from the Mods menu.

## Notes

This is a lightweight vanilla-style crafting mod made for Build 42.
The advanced version currently uses `SkillRequired` rather than `AutoLearn`, since that was the stable working solution during testing.
